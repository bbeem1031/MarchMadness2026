## GitHub Update Instructions

### After Round 1 Results (March 19-20):
```bash
# 1. Pull latest
git pull origin main

# 2. Update files with round 2 picks and results
# Edit: 2026_NCAA_Bracket_Picks_FINAL.pdf (add R2 picks)
# Edit: 2026_March_Madness_Betting_Picks.pdf (add R2 betting)
# Edit: UPSET_INTELLIGENCE.md (add results section)
# Edit: index.html (update live alert bar with results)

# 3. Stage all changes
git add .

# 4. Commit with descriptive message
git commit -m "Update: R1 results + R2 picks added - [date]"

# 5. Push to GitHub
git push origin main
```

### Upload New PDF Files:
```bash
# If rebuilding PDFs, replace existing files:
cp new_file.pdf 2026_NCAA_Bracket_Picks_FINAL.pdf
git add 2026_NCAA_Bracket_Picks_FINAL.pdf
git commit -m "Updated bracket picks with R2 analysis"
git push origin main
```

### Quick Upload via GitHub Web UI:
1. Go to `https://github.com/bbeem1031` → your repository
2. Click **Add file** → **Upload files**
3. Drag and drop the updated PDF/HTML files
4. Scroll down → add commit message: `"Update R[X] picks — [date]"`
5. Click **Commit changes**

### To Remove a File (e.g., old V3 PDF):
```bash
git rm 2026_NCAA_Bracket_Picks_V3.pdf
git commit -m "Remove superseded V3 picks file"
git push origin main
```
